# 🎂 Sweet Bills — Cake Store Billing System

A full-stack web-based billing system designed for cake and pastry stores.
Built with vanilla HTML, CSS, and JavaScript on the frontend and Python Flask
on the backend, with a JSON file as the database.

## 🔧 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Python Flask
- **Database:** JSON (data.json)

## ✨ Features
- 🔐 Secure login system with session-based authentication
- 🛒 Interactive cart — add, remove, and adjust item quantities
- 🧾 Auto-generated printable bill receipts with unique Bill IDs
- 💰 Automatic tax (5%) and custom discount calculation
- 💳 Multiple payment methods — Cash, Card, Online
- 📋 Bill history page with search and revenue summary
- 🎂 25+ products across 5 categories: Cakes, Pastries, Cupcakes, Muffins & Special
- 📊 Real-time dashboard showing today's bills and total revenue

## 🚀 How to Run
1. Clone the repository
2. Install dependencies: `pip install flask`
3. Navigate into the project folder: `cd cake_billing`
4. Run the server: `python app.py`
5. Open your browser at: `http://localhost:5000`
